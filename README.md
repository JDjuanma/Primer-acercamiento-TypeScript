## Gestión de Empleados – CRUD por Consola (TypeScript)

Este es mi primer proyecto realizado con TypeScript, donde implemento un sistema CRUD por consola para gestionar empleados.
El objetivo es aprender la base del lenguaje, manejo de módulos, async/await, entrada por consola, clases y organización del proyecto.

## Tecnologías usadas

TypeScript

Node.js

Módulo readline para entrada por consola

Programación orientada a objetos

npm scripts para compilar y ejecutar
